## üì± Using the Patient Selection System

### Understanding the Three Tabs

Each screening type has three tabs for patient selection:

#### **Tab 1: School Screening Students** üè´
- **What it shows**: Students who have already completed school screenings
- **Best for**: Following up on students identified during school programs
- **How to use**:
  1. Use the search bar to find specific students
  2. Filter by school or grade level
  3. Click on a student card to select them

#### **Tab 2: Manual Registration** ‚úçÔ∏è
- **What it shows**: Patients you manually register in the system
- **Best for**: New patients not in the school system
- **How to use**:
  1. Click "Add New Patient" button
  2. Fill in the patient information form
  3. Submit to create the patient record
  4. The patient will appear in your selection list

#### **Tab 3: Citizen Card Reader** üí≥
- **What it shows**: Patients registered via ID card scanning
- **Best for**: Quick patient identification using government IDs
- **How to use**:
  1. Insert the patient's ID card
  2. System automatically reads patient data
  3. Verify the information is correct
  4. Confirm to select the patient

## üè• Conducting the Screening

### Step-by-Step Workflow

#### **Step 1: Patient Selection**
- Choose patient from one of the three tabs
- Verify patient information
- Confirm parent consent if required

#### **Step 2: Vision Assessment**
- **Distance Vision**: Test each eye separately
- **Near Vision**: Test reading vision
- **Color Vision**: Use color vision test cards
- **Depth Perception**: Conduct depth perception tests

#### **Step 3: Results Recording**
- Enter all test results accurately
- Add any notes or observations
- Record recommendations for follow-up

#### **Step 4: Glasses Assessment** (Mobile Screening Only)
- Determine if glasses are needed
- Record prescription details
- Fit glasses if available
- Schedule delivery if needed

#### **Step 5: Completion**
- Review all entered information
- Complete the screening session
- Export data if needed

## üìä Exporting Your Data

### CSV Export (Available After Screening Completion)

#### **When Export is Available**
- Export button appears **only after** screening is completed
- Button is located next to the "Complete Screening" button
- Available for all screening types

#### **What Gets Exported**
- **Patient Information**: ID, name, DOB, school, grade
- **Screening Results**: All vision test results
- **Assessment Data**: Overall assessment and recommendations
- **Follow-up Information**: Required follow-up actions and dates

#### **How to Export**
1. **Complete** the screening workflow
2. **Look for** the "Export to CSV" button
3. **Click** the export button
4. **File downloads** automatically to your device
5. **Filename format**: `screening-type-patient-name-date.csv`

### Medical Reports Export

#### **Accessing Reports**
1. Go to **Medical Reports** section
2. Select **Export Data** tab
3. Choose export format (CSV, PDF, Excel)
4. Apply filters if needed
5. Click export button

#### **Available Report Types**
- **Screening Reports**: All screening data
- **Vision Reports**: Vision-specific assessments
- **Health Reports**: Overall health data
- **Analytics Reports**: Statistical summaries
